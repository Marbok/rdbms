## Классы анализа
1. Row - строка с данными в таблице
2. Column - столбец в таблице
3. Type - тип данных в столбце (enum?)
4. TableMetaData - информация о данных в таблице (удален)
5. Table - набор строк + мета данные
6. DBMetaData - информация о таблицах в бд (удален)
7. Database - набор таблиц + мета данные

8. Operation и производные (Select, Update) - операция в базе данных (удален)
9. Condition и производные (More, Less) - часть Operation, показывает как выбирать данные (удален)
10. Join - часть Operation операция для объединения (удален)

11. SqlInterpreter - умеет преобразовывать sql запрос в операцию (удален)
